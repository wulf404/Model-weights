**Назначение**
В этой папке находятся ONNX-веса модели детекции в нескольких вариантах.
Они отличаются размером модели и размером входного изображения.

quadros.onnx

quadros_1920.onnx
quadros_2560.onnx

----------------------------------------
**Как читать название**
----------------------------------------
Имя файла обычно состоит из:
1. варианта модели
2. размера входа

Примеры:
- quadron_1280.onnx  -> вход 1280x1280
- quadros_1920.onnx  -> вход 1920x1920
- quadron_4000.onnx  -> вход 4000x4000

Если числа в названии нет, значит это базовый экспорт модели.
Перед использованием лучше проверить входной размер отдельно.

----------------------------------------
**Варианты моделей**
----------------------------------------
- quadron  — самый легкий и быстрый вариант
- quadros  — средний вариант
- quadrom  — более тяжелый вариант

Обычно:
- чем легче модель, тем выше скорость
- чем тяжелее модель, тем выше требования к устройству

----------------------------------------
**Что брать для старта**
----------------------------------------
Для первого запуска рекомендуется:
- quadron_1280.onnx  — если важна скорость
- quadros_1280.onnx  — если нужен баланс

----------------------------------------
**Рекомендации**
----------------------------------------
- не переименовывать файлы без необходимости
- в коде явно указывать, какой вес загружается
- отдельно фиксировать результаты тестов:
  FPS, задержку, качество детекции и размер входа
